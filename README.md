# LSPosed-root-modle

LSPosed Root 模块合集（需 Root + Magisk/LSPosed + Shizuku）

## 模块列表（10个）

| # | 模块 | 功能 |
|---|------|------|
| 1 | ShizukuSceneFix | 修复 Scene 在 Shizuku 授权列表不显示 |
| 2 | GameUnlockerPro_Root | 游戏性能（帧率/温控/GPU/内核调优/机型伪装） |
| 3 | PrivacyGuard_Root | 隐私保护（设备ID/位置/Magisk overlay持久化属性） |
| 4 | AdBlockerX_Root | 广告拦截（系统hosts/PrivateDNS/DNS缓存刷新） |
| 5 | BatteryOptimizer_Root | 省电优化（Doze/冻结/CPU/zram优化） |
| 6 | StepModifier_Root | 步数修改（系统级传感器/数据库注入） |
| 7 | NotifyMaster_Root | 通知管理（全局监听/跨APP过滤/通知队列） |
| 8 | AudioBoost_Root | 音量增强（AudioFlinger/AudioPolicy/全局音量） |
| 9 | VipUnlocker_Root | VIP解锁（伪装机型/Google授权/Magisk overlay持久化） |
| 10 | VideoSaver_Root | 短视频去水印（系统下载服务/媒体扫描） |

## 使用方式

1. 安装 Magisk + LSPosed + Shizuku
2. 下载模块 APK 安装
3. 在 LSPosed 管理器勾选作用域
4. Shizuku 授权 adb 级权限
